# test-prioritization

## Usage:
```bash
Usage: test-prioritization

Options:
  -t, --test-case-folder <TEST_CASE_FOLDER>
  -d, --distance-csv-output-path <DISTANCE_CSV_OUTPUT_PATH>
  -p, --prio-csv-output-path <PRIO_CSV_OUTPUT_PATH>
  -h, --help                                                 Print help
```

## Example 
```bash
$ test-prioritization -t Mozilla_TCs -d ./distance.csv -p ./prio.csv
```
