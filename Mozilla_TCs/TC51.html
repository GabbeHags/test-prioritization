Test Case Description for surf-all-bookmarks

Purpose

This basic testcase is designed to verify that all the bookmarks within the default bookmarks file that is packaged with the browser are valid links.

Initial Conditions

You must have the default bookmarks file installed. Here is the Netscape bookmarks file, here is the Mozilla bookmarks file.

Steps/Description

Using the Bookmarks menu or the Manage Bookmarks window, select and surf each and every bookmark in turn.

Expected Results

Each bookmark should load completely in the browser window. There should be no "dead" links.
