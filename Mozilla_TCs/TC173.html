Test Case Description for page setup (Mac only)

Purpose

Determine that Page Setup dialog comes up properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Do "File | Page Setup"

Expected Results:

Native Page Setup dialog should come up
