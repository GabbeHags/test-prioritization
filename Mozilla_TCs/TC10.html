Test Case Description for Netscape Technical Support help content

Purpose

Determine whether Netscape Technical Support displays correctly

Initial Conditions

Netscape should be started up

Steps/Description

  1. From the Help Drop Down Menu, select Help and Support Center
  2. Click the Assisted Support Options link
  3. Click the Communicator 4.x link
  4. Click on the Netscape link
  5. Click on the Intertional Browsers link

Expected Results

1. The Help and Support Center page should come up

2. You should be taken to the Need some help? page

2. Help number should be listed 1(800)411-0707 (They will help you for low-cost assistance)

2. There should be links to Communicator 4.x, Netscape 6, International Browsers

3. You should be taken to a general Help page for Communicator 4.x

4. You should be taken to a general Help page for Netscape 6

4. Verify that the browser mentioned on this page is the correct version of the Netscape 6 browser

5.You should be taken to a general Help page for International Support
