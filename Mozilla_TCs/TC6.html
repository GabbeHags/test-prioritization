Test Case Description for PageUp/PageDn keys in Help

Purpose

This basic testcase is designed to verify that the Page Up and Page Down Keys function properly in help

Initial Conditions

Netscape should be started up

Steps/Description

  1. Select the toplevel Help menu.
  2. Select Technical Support
  3. Push the Page Down Key on your keyboard
  4. Push the Page Up Key on your keyboard
  5. Click a few random links in help and try using these keys again

Expected Results

2. You should be taken to http://help.netscape.com/

4. This should move you down the page until the end

5. This should move you up the page until you reach the top

6. Page up and Page Down keys should consistantly move you up and down pages
