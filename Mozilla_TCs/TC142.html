Test Case Description for remove all saved passwords

Purpose

Verify that you can remove all username and password information and it will no longer autofill nor will anything be listed in password manager

Initial Conditions

Password manager should be enabled and you should have saved username and password information for several sites

Steps/Description

  1. From the edit drop down menus, select preference
  2. Click on the arrow next to advanced
  3. Click on passwords
  4. Click View stored passwords
  5. Click remove all
  6. Click cancel
  7. Click View saved passwords
  8. Click remove all
  9. Click ok
  10. Click View saved passwords

Expected Results

1. Preference pane should open

2. Advanced options should appear

3. Password manager options should appear on right pane

4. A list of saved passwords should appear with your saved sights and usernames appearing in the list

5. All of the saved site and username information should be removed from the list

7. All of the saved site and username information should again, appear in the list

8. All of the saved site and username information should be removed from the list

10. There should be no information listed under the passwords saved tab

11. This site and username should no longer appear on the list of saved passwords

13. This site should be removed from the list
