Test Case Description for dragging URLs onto the desktop

Purpose

Confirm that dragging a link from the browser window onto the desktop creates a shortcut, linking to the correct URL.

Steps/Description

  1. Open the testfile.html test file
  2. Drag the Drag Me link onto the desktop
  3. Open the shortcut created in the last step
  4. Drag the Space Shuttle link onto the desktop
  5. Open the shortcut created in the last step

Expected Results

2. A link to the dragme.html file should be created

3. The dragme.html file should open the browser window

4. A link to the large space shuttle image should be created

5. The large space shuttle image should open in the browser window
