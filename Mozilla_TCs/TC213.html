Test Case Description for listing available remote panels

Purpose

Determine the list of available remote panels

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. From the Sidebar, select "Tabs | Customize Sidebar"
  3. From the window that opens up, the pane titled "AvailablePanels" should list the following folders by default: Favorites, From Netscape, From Others, International.
  4. Click and open each folder mentioned above to see that they are not empty and that a listing of panels is available in each folder
  5. Verify that the Favourites folder lists these panels: CNN.com, Horoscope, Mozillazine, News, Stocks, Tech News, Weather
  6. Verify that the From Netscape folder lists these panels: Calender, Horoscopes, Local Events, Local Movies, Local News, News, Sports, Stocks, Tech News, Weather
  7. Verify that the From Others folder lists these panels: BackFlip Buddy, CNET.com, CNN.com, eBay, Kinderview, Mapquest, Mozillazine, Red Herring, Webstakes.com and the folder My Netscape Network
  8. The International folder should list the following folders: Brazil, Canada, France, Germany, Japan, Portugal, Spain, Sweden, UK

Expected Results

All default remote panels should be available
