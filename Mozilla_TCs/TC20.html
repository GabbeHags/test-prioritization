Test Case Description for verifying theme items functionality (2)

Purpose

To verify all the selectable items in Classic theme work properly on different applications under browser like composer, mail, news, aim, address book, etc.

Description

  1. From the main menu bar, select Edit > Preferences > Appearance > Themes.
  2. Select "Classic" theme on the right hand side of the dialog.
  3. Click the "Apply Classic" button.
  4. Close the Preferences dialog.
  5. Click through all the items on menu bars and tool bars under composer, mail, news, aim, and address book to see all the look and feel are consistent.

Expected Results

All the selectable items and icon under browser should work as advertised
