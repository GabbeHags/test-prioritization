Test Case Description for custom-keywords

Purpose

This basic testcase is designed to verify that the edit properties feature works as prescribed for various items.

Initial Conditions

  • You must have a bookmarks file.
  • To promote consistency, you may choose to use the default set of Netscape Seamonkey Bookmarks, which I use. I have also provided the default Mozilla bookmarks file.
  • You also have the option to use this very large bookmarks file.
  • You must also customize your sidebar to include a bookmarks panel.

Steps/Description

  1. Select Bookmarks|Manage Bookmarks.
  2. Select a bookmark. Select File|Bookmark Properties.
  3. Enter the word "blah" in the custom keyword field. Click "OK".
  4. In a browser window. Type "blah" into the url field and press enter.

Expected Results

  1. You should be taken to the url associated with the same bookmark the custom keyword "blah" was associated with.
  2. The browser should NOT perform an "internet keyword search", "netsape smart browsing search", or any other search.
