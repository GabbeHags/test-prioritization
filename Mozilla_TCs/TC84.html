Testcase for Form Manager Pop-up

Purpose

Verify that a form fill popup menu is activated when a field gets accessed that has been saved in Form Manager data.

Initial Conditions

N/A

Steps/Description

  1. Go to a site that requires you to fill in data such as ivillage.com (then go to the page to create an account)
  2. Fill in data on the form
  3. From the Edit drop down menus, select save form data
  4. Shut down the browser
  5. Start the browser again and return to ivillage.com
  6. Click in the first field

Expected Results

3. A dialog box should appear, alerting you to the fact that the data has been captured

6. A popup list should appear with the choice of selecting what you typed in the first time
