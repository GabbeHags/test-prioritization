Testcase for cancel after deleting data from form manager

Purpose

Verify that if you delete data in the Form Manager then click cancel, the data has not been deleted

Initial Conditions

You should have no form data saved.

Steps/Description

  1. Go to a site that needs information, for example tripod.com and then click signup
  2. Fill in all the fields on the form
  3. From the Edit menu, select Save Form Info
  4. From the Tools menu, open the Form Manager sub-menu, then choose Edit Form Info
  5. Delete several of the values in the Form Manager (for instance Last: in Name and email in address)
  6. Click Cancel
  7. From the Tools menu, open the Form Manager sub-menu, then choose Edit Form Info
  8. Verify that the previously deleted information is still in Form Manager

Expected Results

3. A dialog should appear saying data has been captured and the data from this form should be saved to the form manager

4. Form manager should appear and your saved data should be listed

7. Your data values should not have been changed

8. Your previously deleted information should still be in Form Manager
