Test Case Description for remove all never saved passwords

Purpose

Verify that you can remove all site and username information from the passwords never saved section of password manager

Initial Conditions

Password manager should be enabled and you should have saved username and password information for several different sites

Steps/Description

  1. From the tasks drop down menu, select privacy and security ->password manager->view stored passwords
  2. Click Passwords never saved tab
  3. Click remove all
  4. Click cancel
  5. Click View saved passwords
  6. Click the passwords never saved tab
  7. Click remove all
  8. Click ok
  9. From the tasks drop down menu, select privacy and security ->password manager->view stored passwords
  10. Click Passwords never saved tab

Expected Results

1. A list of saved passwords should appear

2. A list of passwords never saved should appear

3. All items in the list of passwords never saved should be removed

4. Window should close

6. All items in the list of passwords never saved should, again, be on the list

7. All items should be removed from the list

10. There should be nothing listed below
