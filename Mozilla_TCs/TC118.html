Test Case Description for multiple, differing animated images

Purpose

To determine whether multiple, differing animated images on the same page are properly displayed.

Initial Conditions

None.

Steps/Description

  1. Display the attached web pages
  2. (if site still present, visit http://www.animationcity.net/animcity.htm, as an additional stress test case.)

Expected Results

All animated images should animate smoothly, without stopping, or displaying any glitches. (Communicator 4.6x or IE can be used for a more specific description of the expected results.)

Animated GIFs off the net occasionally have a pre-specified number of iterations, normally 1000, and will halt animation deliberately.
