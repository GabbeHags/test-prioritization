Test Case Description for using classic theme

Purpose

To verify the classic theme is displayed.

Description

  1. From the main menu bar, select Edit > Preferences > Appearances > Themes.
  2. Select "Classic" theme on the right hand side of the dialog.
  3. Click the "Apply Classic" button.
  4. Close the Preferences dialog.
