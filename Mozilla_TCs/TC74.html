Test Case Description for dragging from the Find File window

Purpose

Verify that files, folders, or shortcuts dragged from a Find File window opens properly in the browser window (Windows / Mac OS)

Steps/Description

  1. Download the test archive for your system, linked below, to your computer and decompress it.
  2. Open your system's find application.

    In Windows, open the Start menu, open the Find sub-menu, and choose Files and Folders.

    In Mac OS, switch to Finder, and choose Find from the File menu.

  3. Search the disk volume for file mozqahome.html. Drag the file icon from the Find window to your browser window.
  4. Search the disk volume for find-folder. Drag the folder icon from the Find window to your browser window.
  5. Search the disk volume for dmozBookmark. Drag the shortcut file icon from the Find window to your browser window.

Expected Results

3. The browser should display the text of the Mozilla QA Home Page.

4. The browser should display the directory contents of find-folder.

5. The browser should display the dmoz.org site.
