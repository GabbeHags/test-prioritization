Test Case Description for view never saved password (task)

Purpose

Verify that when you have designated not to ever save a username and password in password manager, this information can be viewed in the password manager through tasks

Initial Conditions

Browser should be started, password manager should be enabled and you should have a username and password for a site, such as http://www.nbci.com

Steps/Description

  1. Go to a sight that requires a password, such as http://www.nbci.com
  2. Enter a username and password and click sign in to nbci.com
  3. You should be presented with a dialog box asking if you would like to save your username and password, click never for this site
  4. From the Tasks drop down list, select privacy and security->password manager->View stored passwords
  5. Click the passwords never saved tab
  6. Click passwords
  7. Click the view stored passwords button
  8. Click the Passwords never saved tab
  9. Repeat for several different sites

Expected Results

3. This site should be saved in the passwords Never Saved portion of password manager

4 The password manager box should come up with the passwords saved tab in front

5. You should see www.nbci.com listed
