Test Case Description for save-state

Purpose

This basic testcase is designed to verify that bookmarks are added and deleted from the personal

Initial Conditions

  • You must have a bookmarks file.
  • To promote consistency, you may choose to use the default set of Netscape Seamonkey Bookmarks, which I use. I have also provided the default Mozilla bookmarks file.
  • You also have the option to use this very large bookmarks file.
  • You must also customize your sidebar to include a bookmarks panel.

Steps/Description

  1. Select Bookmarks|Manage Bookmarks.
  2. Click on disclosure triangles to open/close several folders.
  3. Remember which were open/closed.
  4. Do the same for bookmarks folders in the bookmarks sidepanel. (Do not close Manage Bookmarks)
  5. Quit the browser (alternately, induce a crash)
  6. Launch the browser and open the sidebar and Manage Bookmarks.

Expected Results

  1. The bookmarks tree should have the same shape, ie folders that were closed should still be closed and those that were open should still be open.
  2. This should be true for the bookmarks in the sidepanel as well.
