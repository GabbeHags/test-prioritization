Test Case Description for remove saved password preferences

Purpose

Verify that you can remove a username and password for a given site and it will no longer autofill on that sight nor will it be listed in password manager

Initial Conditions

Password manager should be enabled, in a different browser create an account at a site that requires a username and password such as http://www.espn.com

Steps/Description

  1. Go to a site that you have not been to before in Netscape 6, such as http://www.espn.com
  2. Enter your username and password
  3. When a dialog appears asking if you would like to save this information, click yes
  4. From the edit drop down menus, select preference
  5. Click on the arrow next to advanced
  6. Click on passwords
  7. Click View saved passwords
  8. Highlight this sight and username in the list and click remove
  9. Click cancel
  10. Click View saved passwords
  11. Highlight this sight and username in the list and click remove
  12. Click ok
  13. Click View saved passwords

Expected Results

3. Your username and password should be saved to the password manager

4. Preference pane should open

5. Advanced options should appear

6. Password manager options should appear on right pane

7. A list of saved passwords should appear with your saved sight and username appearing in the list

8. This site's information should no longer appear in the list

9. Window should close

10. This site should once again, be in the list of saved passwords

11. This site and username should no longer appear on the list of saved passwords

13. This site should be removed from the list
