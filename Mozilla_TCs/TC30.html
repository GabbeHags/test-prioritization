Test Case Description for copy-paste-items

Purpose

This basic testcase is designed to verify that copy and paste function properly.

Initial Conditions

  • You must have a bookmarks file.
  • To promote consistency, you may choose to use the default set of Netscape Seamonkey Bookmarks, which I use. I have also provided the default Mozilla bookmarks file.
  • You also have the option to use this very large bookmarks file.
  • You must also customize your sidebar to include a bookmarks panel.

Steps/Description

  1. Select Bookmarks|Manage Bookmarks.
  2. Select any item.
  3. Select Edit|Copy.
  4. Select any item.
  5. Select Edit|Paste.
  6. Cycle through the 3 possible choices in each of step(2) and step(4). There are nine permutations.

Expected Results

  1. Every item should be copyable(bookmark, folder, separator).
  2. The copied item should be pasted below the selected item in each case.
  3. For each paste the newly pasted items should appear in the 4 standard locations.
