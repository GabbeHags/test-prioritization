Test Case Description for paragraph style printing

Purpose

Determine that Seamonkey prints pages with different paragraph styles properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to open first paragraph styles test page.
  3. After each test page opens, select "FILE | PRINT" menu option to print the test.
  4. Click on "Next Test" at bottom of the page to proceed to next test.

Expected Results

All pages should print as they appear on the screen.
