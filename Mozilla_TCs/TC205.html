Test Case Description for bookmarks manager

Purpose

Determine that changes reflect immediately from bookmarks manager

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. Add the Bookmarks panel to the Sidebar using the "Customize Sidebar" option
  3. Open the Bookmarks panel in the Sidebar
  4. Visit any url, say, www.yahoo.com
  5. Bookmark that link using "Bookmarks | Add Current Page"

Expected Results

As soon as you bookmark the page in step 5, the change should get reflected in the Bookmarks panel in the Sidebar too and you should see the added bookmark in the panel
