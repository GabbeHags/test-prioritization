Test Case Description for Default Panels

Purpose

Determine that default panels and their contents load in Sidebar

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. In the Sidebar, verify that the following default panels appear:
    *Search
    *Buddy List
    *Stocks
    *News (open)
    Shopping
    Today's Tips
    Bookmarks
  3. Also verify the content in each default panel appears by selecting them one by one

*: Default open panels.

Expected Results

All the default panels listed above should appear in the Sidebar and their content should be proper.
