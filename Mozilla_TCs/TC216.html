Test Case Description for What's Related lists

Purpose

Determine that What's Related lists lists related sites

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. Visit any url, say, www.yahoo.com
  3. From the Sidebar, select the "What's Related" tab to open it (This tab gets added by default)
  4. Observe that a listing of websites is given in the What's Related tab

Expected Results

What's Related tab should list a number of related sites to yahoo.com, eg: aol.com, google.com, msn.com, etc.
