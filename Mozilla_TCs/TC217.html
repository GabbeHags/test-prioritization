Test Case Description for What's Related links

Purpose

Determine that links in What's Related panel work

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. Visit any url, say, www.yahoo.com
  3. From the Sidebar, select the "What's Related" tab to open it (This tab gets added by default)
  4. Observe that a listing of websites is given in the What's Related tab
  5. Double click with the mouse on any of the links
  6. The selected link should open in the browser window. Try a few more links from the panel

Expected Results

Appropriate links should open in the browser window when clicked from the What's Related panel in the Sidebar
