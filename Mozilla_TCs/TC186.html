Test Case Description for url-open-sidebar

Purpose

This basic testcase is designed to verify that initiating a search in the URL bar triggers the opening and loading of search results into the Sidebar Search tab.

Initial Conditions

  • Open a new browser window.
  • All prefs should be set to their defaults (new profile).

Steps/Description

  1. Type a multi-word search term in the URL bar. Anything will do, "Claudius 'CMaximus' Gayle" for example.
  2. Click the "Search" Button.

Expected Results

  1. After search results load in the browser window, the Sidebar should pop open to the Search tab and be populated with search results as well.
  2. The textfield in the sidebar search tab should be populated with your search term exactly as you typed it.
