Test Case Description for testing url-do-search

Purpose

This basic testcase is designed to verify that clicking the "Search" button with an altered URL Bar performs a search on the altered text.

Initial Conditions

Open a new browser window.

Steps/Description

  1. Type something into the URL Bar or edit already existing contents.
  2. Click the "Search" button to the right of the URL Bar.

Expected Results

  1. Clicking the "Search" button with a "dirty" URL bar should launch a search on the terms in the URL bar.
  2. You should see a page with search results in the browser content window. *there are MySidebar ramifications with this testcase that are treated elsewhere.
