Test Case Description for listing available local panels

Purpose

Determine the list of available local panels

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. From the Sidebar , select "Tabs | Customize Sidebar"
  3. From the window that opens up, the pane titled "AvailablePanels" should list the following local panels by default: What's Related, Search, Bookmarks

Expected Results

All default local panels should be available
