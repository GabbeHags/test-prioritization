Test Case Description for show/hide toolbar in Help

Purpose

This basic testcase is designed to verify that it is possible to hide the toolbar in help, as well as open it again

Initial Conditions

Netscape should be started up

Steps/Description

  1. Select the toplevel Help menu.
  2. Select Help Contents
  3. Click the arrow on the upper right hand corner
  4. Click on the smaller arrow that now appears on the upper left hand corner

Expected Results

2. You should be taken to the help contents page

3. The toolbar should collapse

5. The toolbar should now appear with buttons for forward, back, refresh and print
