Test Case Description for dragging items to the Bookmark Manager

Purpose

Determine whether the Bookmark Manager properly handles items dragged onto it.

Initial Conditions

Download the test archive for your system, linked below, to your computer and decompress it.

Steps/Description

  1. Download the test archive for your system, linked below, to your computer and decompress it.
  2. On Windows, drag the Netcenter file to the to the Bookmark Manager
  3. On Mac OS, drag the Mac OS bookmark and Netscape Bookmark files to the Bookmark Manager
  4. Drag a disk icon (e.g. Macintosh HD on Mac OS, or C:\ on Windows) to the Bookmark Manager
  5. Simultaneously drag the mozilla.gif, mozilla.jpg and OwlAlpha.png files to the Bookmark Manager
  6. Drag mozillazine.html file to the Bookmark Manager
  7. Drag zinealias.html shortcut (alias) to the Bookmark Manager
  8. (Mac OS) Drag the contents of the Stress Tests folder simultaneously to the Bookmark Manager

Expected Results

2. A URL pointer to the hard drive should be added.

3. Bookmark links to each of the 3 image files should be created.

4. A bookmark to the mozillazine.html file should be created

5. A second bookmark to the mozillazine.html file should be created

6. (Mac OS) Bookmarks should be created for /. (slashdot.org) and dmoz%3A.org (dmoz.org).

  • bookmarks.sit (Mac OS)
