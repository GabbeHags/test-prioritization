Test Case Description for printing HTML widgets

Purpose

Determine that Seamonkey prints pages with HTML widgets properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to view a page with HTML widgets
  3. Select menu option "FILE| PRINT"
  4. Press "OK" to print the page

Expected Results

After Step 4, all the HTML widgets should print as seen on screen
