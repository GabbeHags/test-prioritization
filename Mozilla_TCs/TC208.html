Test Case Description for adding panel using javascript

Purpose

Determine that a panel can be added using javascript to the Sidebar

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. Click here to open a page to add the bugzilla panel to the Sidebar using javascript
  3. Click on the link "Add Bugzilla panel to your sidebar" on the page
  4. Press "OK" on the confirm dialog
  5. Bugzilla panel should get added to the Sidebar at the bottom
  6. Click on the panel to open it

Expected Results

Panel should get added by javascript
