Test Case Description for cursor icon

Purpose

Determine that the mouse cursor should change to the proper icon when moved over links

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. From the Sidebar, click on any of the default panel,say, CNN.com and open it
  3. Moving the mouse over the links in the panel should change the mouse icon thus making the user understand that the text is clickable
  4. Moving the mouse over plain html text should also change the mouse icon
  5. Moving the mouse away from the clickable text(to a blank space) should change the mouse icon again

Expected Results

Mouse icons should change properly
