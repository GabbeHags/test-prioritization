Test Case Description for image loading (local)

Purpose

To confirm that images load directly into the web browser from a local hard drive.

Initial Conditions

Disk and memory caches to be cleared.

Steps/Description

  1. Copy these below-linked files to your hard drive: vlpan12.gif, vlpan11.jpg and wall_ni.png.
  2. View each of those images.

Expected Results

After loading vlpan12.gif, a 530x462 GIF image of the surface of Mars should display.

After loading vlpan11.jpg, a 1496x1536 JPEG image of the surface of Mars should display.

After loading wall_ni.png, a 640x480 PNG image of the Great Wall should display.
