Test Case Description for drag receipt in Help

Purpose

This basic testcase is designed to verify that it is not possible to drag an html document to the help window

Initial Conditions

Netscape should be started up

Steps/Description

  1. Save the file below to your desktop
  2. Select the toplevel Help menu.
  3. Select Help Contents
  4. Drag the file from your desktop into the help windowr

Expected Results

3. You should be taken to the help contents page

4. You should not be able to do this
