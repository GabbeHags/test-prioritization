Test Case Description for Mozillazine result page printing

Purpose

Determine that Seamonkey prints Mozillazine Poll Result page properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to view the mozillazine polls result page
  3. Select menu option "FILE | PRINT"
  4. Press "OK" to print the page

Expected Results

After Step 4, the polls result page should print as is.
