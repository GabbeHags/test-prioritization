Test Case Description for non-default width

Purpose

Determine that non-default width of Sidebar is retained after new window is opened or browser is restarted

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. Click with the mouse on the Sidebar grippy(mouse pressed) and expand the sidebar
  3. Make the width of the Sidebar to whatever you want (upto max allowed)
  4. Select File| New Navigator Window to open a new browser window
  5. Observe that the Sidebar width should be the same as in the earlier window and should not appear changed
  6. Also, after step3, select File|Quit to close the browser
  7. Restart the browser

Expected Results

After step 3 and step 6, sidebar non-default width should be the same as changed earlier
