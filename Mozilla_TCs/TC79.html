Test Case Description for dragging URLs across browser windows

Purpose

To confirm that dragging a text or image link from one browser window to another opens the link in the target window.

Steps/Description

  1. Open the testfile.html test file
  2. Open a new browser window
  3. Drag the Drag Me link from the first browser window to the new window
  4. Drag the Space Shuttle image from the first browser window to the new window.

Expected Results

3. The dragme.html page should open in the new window

4. The large space shuttle image should open in the new window.
