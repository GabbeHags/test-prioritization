Test Case Description for printing page source

Purpose

Determine that Seamonkey prints page source properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Go to mozilla home page
  3. Select menu option "View | Page Source" to view the source code
  4. A new window with the source opens up.
  5. Select menu option "FILE | PRINT"
  6. Press "OK" to print the page

Expected Results

After Step 6, the page source should print as seen on screen
