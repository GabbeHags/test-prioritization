Test Case Description for anchor history

Purpose

This testcase is designed to verify that anchored links are properly added to the global history file.

Initial Conditions

Make the window short enough so that you can only see the Steps/Description portion of this document.

Steps/Description

  1. Click on this anchor link to the purpose.
  2. Click on this anchor link to the steps.
  3. Click on this anchor link to the links.
  4. Click "Back" three times. Click forward three times.
  5. Select Go|History.

Expected Results

  1. You'll see the view scroll as you follow each link, the url should be updated with the #<anchorname> appended to the url.
  2. Step(4) should load the pages in this order; #steps, #purpose, main page, #pupose, #steps, #links.
  3. The Global History window should list each named anchor as a separate entry.
