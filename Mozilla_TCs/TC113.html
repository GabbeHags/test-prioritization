Test Case Description for image resize to original size

Purpose

Determine that it is possible to resize an image to it's original size after it has been resized to window size.

Initial Conditions

Have mozilla started up

Steps/Description

Complete these steps for all images below

  1. Open each of the files listed below
  2. Right click the image and select resize to window
  3. Right click again and select resize to original

Expected Results

Image should resize to it's original size.
