Testcase for troubleshooting help about "can't see anyone in buddy list"

Purpose

Verify that there is Troubleshooting Help about the problem "if you can't see anyone in your buddy list window."

Initial Conditions

Mozilla should be started up

Steps/Description

  1. From the Help drop down menu, select Help and Support Center
  2. Open your sidebar if it is not already open (click on the widget to the left hand side of your browser)
  3. Click on the Index tab
  4. Doubleclick on the T in the index
  5. Click the If You Can't See Anyone in you Buddy Window

Expected Results

1. The Help and Support Center window should open

3. An alphabetic list should display on the left hand side of the window

4. The T should expand and you should see Troubleshooting listed

5. You should be taken to Help information regarding Buddy List Window issues
