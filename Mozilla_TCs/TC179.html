Test Case Description for table printing.

Purpose

Determine that Seamonkey prints Tables properly

Initial Conditions:

None

Steps/Description

  1. Launch Apprunner
  2. Open first tables test page
  3. After each test page opens, select "FILE | PRINT" menu option to print the test.
  4. Click on "Next Test" at bottom of the page to proceed to next test.

Expected Results

All tables should print as they appear on the screen.
