Test Case Description for printing pages with alignment

Purpose

Determine that Seamonkey prints pages with alignments properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to view a page with aligned text
  3. Select menu option "FILE| PRINT"
  4. Press OK on Print Dialog Box to print the page

Expected Results

After Step 4, the page should be printed as seen on screen
