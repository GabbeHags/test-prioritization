Test Case Description for printing lists

Purpose

Determine that Seamonkey prints Lists properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to open a page with different types of lists.
  3. Select "FILE | PRINT" menu option to print the page.

Expected Results

All lists should print as they appear on the screen.
