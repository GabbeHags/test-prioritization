Test Case Description for dragging from manage bookmarks menu to personal toolbar

Purpose

Verify that you can drag a bookmark from the Bookmarks menu to the Personal Toolbar.

Steps/Description

  1. From the Tasks drop down menu, select Tools-> JavaScript Console
  2. Click Clear and close the console
  3. From the Bookmarks drop down menu select manage bookmarks
  4. Drag a bookmark from the manage bookmarks window to the personal toolbar
  5. Verify that a bookmark icon appears on your toolbar and when clicked, it takes you to the appropriate website
  6. From the Tasks drop down menu, select Tools-> JavaScript Console
  7. Verify that there are no new errors in the console
