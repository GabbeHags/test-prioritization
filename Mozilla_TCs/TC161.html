Test Case Description for CSS2 page printing

Purpose

Determine that Seamonkey prints CSS2 pages properly

Steps/Description

  1. Run the NIST CSS2 Test Suite
  2. Run the NIST CSS1 Tests
  3. Run David Baron's CSS Tests
  4. Run W3C's CSS Tests Suite
  5. Run Ian Hickson's Evil Test Suite
  6. After visiting each test page, print each test (select Print from the File menu)

Expected Results

All CSS pages should print as they appear on screen
