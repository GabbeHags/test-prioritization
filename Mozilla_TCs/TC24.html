Test Case Description for using modern theme

Purpose

To verify the modern theme is displayed.

Description

  1. From the main menu bar, select Edit > Preferences > Appearances > Themes.
  2. Select "Modern" theme on the right hand side of the dialog.
  3. Click the "Apply Modern" button.
  4. Close the Preferences dialog.
