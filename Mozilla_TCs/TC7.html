Test Case Description for Security Center help content

Purpose

Determine whether the Security Center displays correctly

Initial Conditions

Netscape should be started up

Steps/Description

  1. From the Help Drop Down Menu, select Help and Support Center
  2. Click on the Security Center Link

Expected Results

The images should be too big for the browser window and you should have to scroll to see the entire pictures (should not resize with text)

Links

1. Help and Support Center should open

2. Verify you are taken to the Security News page and it lists current Security issues (currently mentions the Sun JVM Issues, Cookie Vulnerability, SmartDownload Exploit, and The Brown Oriffice Vulnerability)
