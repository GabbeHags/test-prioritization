Test Case Description for dragging a large image (stress)

Purpose

Verify that dragging a large (e.g. 5 MB) image across browser windows, to desktop, and into mail message works properly.

Steps/Description

  1. Open the p43886.jpg image file in the browser window
  2. Open a second browser window
  3. Drag the image into the second browser window
  4. Drag the image onto the desktop
  5. Open a HTML Compose Mail window (if the compose window opens in plain text mode, close the window, then in the mail window Ctrl+click (Cmd+click in Mac OS) the Compose button)
  6. Drag the image into the Compose Mail window

Expected Results

2. The second browser window should also open the image.

3. The image file should be saved onto the desktop

5. The image should be copied to the Compose Mail window
