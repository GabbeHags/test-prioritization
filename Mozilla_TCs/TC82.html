Test Case Description for dragging the proxy icon

Purpose

Verify that dragging the URL proxy icon (in the Location Bar) to a browser window opens that URL in target window.

Steps/Description

  1. Open a Web site
  2. Open a new browser window
  3. Drag the URL proxy icon in the Location Bar of the first browser window to the new window

Expected Results

3. The Web site should open in the target window
