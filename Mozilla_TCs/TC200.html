Test Case Description for mouse over panels

Purpose

Determine that when the mouse is moved over panels, the panels highlight

Initial Conditions

This test works only with the "Classic" Theme set for the browser.

Steps/Description

  1. Launch the browser
  2. Move the mouse on any of the panels in the Sidebar
  3. Each panel should highlight making itself visible when the mouse is over it

Expected Results

Panels should highlight when mouse is moved over them
