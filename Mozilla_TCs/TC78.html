Test Case Description for dragging a text file from the desktop

Purpose

Confirm that dragging a text file to the browser window displays the document contents

Steps/Description

Drag a text file from your desktop to the browser window

Expected Results

The document content should be displayed in the browser window as if the file were opened from File|Open.
