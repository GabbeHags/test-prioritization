Test Case Description for image resize after resized window

Purpose

Determine that it is possible to resize an image to it's original size after it has been resized to window size and then you can resize to browser again.

Initial Conditions

Have mozilla started up

Steps/Description

Complete these steps for all images below

  1. Open each of the files listed below
  2. Right click the image and select resize to window
  3. Right click again and select resize to original
  4. Right click and select resize to window

Expected Results

Image should resize to the browser window.
