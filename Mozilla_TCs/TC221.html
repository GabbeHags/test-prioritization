Test Case Description for opening web location dialog

Purpose

Testing access to the web location dialog

Initial Conditions

The browser should already be open.

Steps/Description

Test the following to see if all of these tasks will bring up the Open Web Location dialog:

  1. From the menubar of the browser, select File > Open Web Location.
  2. Use the keyboard shortcut specific to your platform:
    • Win32 and Unix: Control+Shift+L
    • Macintosh: Command+Shift+L

Expected Results

The Open Web Location should be displayed, regardless of the means you used above.
