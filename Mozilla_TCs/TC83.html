Test Case Description for dragging the proxy icon

Purpose

Verify that dragging the URL proxy icon (in the Location Bar) to the desktop creates a shortcut linking to the correct URL.

Steps/Description

  1. Open a Web site
  2. Drag the URL proxy icon in the Location Bar onto the desktop
  3. Close the browser window, and then open the shortcut file created

Expected Results

2. A shortcut to the URL should be created

3. The URL should re-open in the browser window
