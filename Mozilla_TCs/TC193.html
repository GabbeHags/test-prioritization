Test Case Description for adding a panel

Purpose

Determine that a panel can be added sucessfully to the Sidebar

Initial Conditions

This test works only with the "Classic" Theme set for the browser.

Steps/Description

  1. Launch the browser
  2. From the sidebar, select "Tabs | Customize Sidebar"
  3. From the window titled 'Customize Sidebar', select any panel from any of the folders available and click on the 'Tabs' button to add the panel to the sidebar.
  4. Press OK to close the window.

Expected Results

After step 2, the seleced panel should get added to the pane titled "My Sidebar". Pressing the OK button in step3 should close the "Customize Sidebar" window and the newly added panel should appear at the bottom of the Sidebar in the navigator window.
