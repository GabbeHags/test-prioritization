Test Case Description for verifying theme items functionality (1)

Purpose

To verify all the selectable items under browser in Classic theme.

Description

  1. From the main menu bar, select Edit > Preferences > Appearance > Themes.
  2. Select "Classic" theme on the right hand side of the dialog.
  3. Click the "Apply Classic" button.
  4. Close the Preferences dialog.
  5. Click through all the items on menu bars, tool bars, and floating component bars.

Expected Results

All the selectable items and icons under browser should work as advertised.
