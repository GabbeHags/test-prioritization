Test Case Description for copy-complex-page

Purpose

Determine whether complicated rich text (such as massively nested tables) can be copied.

Initial Conditions

None.

Steps/Description

For each of the following links:

  1. Select and copy the page content
  2. View the system Clipboard

(examples from NGLayout Stress tests)

Expected Results

The page content should be properly copied to the Clipboard.
("Proper" is subjective; compare to IE 4.x or 5.x as a baseline and use your best judgement.)
