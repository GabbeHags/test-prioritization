Test Case Description for multiple passwords

Purpose

Verify that when you have saved a username and password for a website and you later change the password, that password manager picks up the new password

Initial Conditions

Password manager should be enable and using a different browser, create a username and passwords for a site such as lycos (http://my.lycos.com) or another of your choice

Steps/Description

  1. Go to a sight that requires a password, such as http://my.lycos.com
  2. Enter a username and password and click sign in
  3. When presented with the save password dialog, click yes
  4. Shut down Netscape 6
  5. In a different browser, return to the site and change your password
  6. Open Netscape 6
  7. Return to http://my.lycos.com (or other site, if used)
  8. Delete the prefilled password information and enter the new password
  9. Click login
  10. Shut down Netscape 6
  11. Open Netscape 6
  12. Return to http://my.lycos.com (or other site, if used)
  13. The password and username information should be prefilled, click sign in

Expected Results

3. The username and password should be saved in Password Manager

9. You should be logged in to the site

13. You should be logged in to the site

9. The username and password should be saved in Password Manager

12. You should be presented with a dialog box that offers a choice between the two usernames

13. You should be signed in to the site
