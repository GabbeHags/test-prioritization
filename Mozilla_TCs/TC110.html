Test Case Description for surf golinks

Purpose

This basic testcase is designed to verify that links generated from session history in the Go menu function properly.

Initial Conditions

Execute the gomenu-list.html testcase first.

Steps/Description

  1. Select Go|Back.
  2. Select Go|Forward.
  3. Select Go|Netcenter.
  4. Click any link on the page.

Expected Results

  1. Each selection should take you to the appropriate page in your session history.
  2. After step(4), the Go menu should only contain "Netcenter" and the title of the page you just surfed to by clicking a link.
