Test Case Description for using modern theme (visual inspection)

Purpose

Visually inspect menu bar, floating component bars and all the tool bars.

Description

  1. From the main menu bar, select Edit > Preferences > Appearance > Themes.
  2. Select "Modern" theme on the right hand side of the dialog.
  3. Click the "Apply Modern" button.
  4. Close the Preferences dialog.

Expected Results

All the menu bars, tool bars, task bars and the widgets are well displayed.
