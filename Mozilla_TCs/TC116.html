Test Case Description for save image

Purpose

Determine that you can right click an image and save to your local drive.

Initial Conditions

Have mozilla started up

Steps/Description

Complete these steps for all images below

  1. Open one of the files listed below
  2. Right click the image and select save image
  3. Browse to where you want to save on your local disk and save.
  4. Open the file in browser (determine the picture has been saved)

Expected Results

Image should be saved to local disk.
