Test Case Description for copy-large-text

Purpose

Determine whether extremely large text blocks can be copied from web pages.

Initial Conditions

None.

Steps/Description

For each of the below-linked text pages:

  1. View the page.
  2. Select all of the text.
  3. From the Edit menu, select Copy.
  4. View the System Clipboard (or copy it into a copy/paste enabled text editor, such as BBEdit or WordPad)

Expected Results

The text should be successfully copied, and displayed in its entirety in the System Clipboard (or within the attempted application).
