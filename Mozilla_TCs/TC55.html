Test Case Description for copy-bad-stuff

Purpose

Determine whether non-copyable page content such as UI widgets are correctly omitted from the Clipboard content.

Initial Conditions

None.

Steps/Description

  1. View the attached test page
  2. Select all of its contents
  3. View the System Clipboard.

    ---

    To view the system Clipboard:
    Mac OS: Select "Show Clipboard" from the Finder.
    Windows: Select "Clipboard Viewer" from the Accessories subgroup of the Programs group.

Expected Results

The System Clipboard content should not contain any of the page's UI widgets (e.g. buttons, check boxes, etc.). However, all text labels should be properly copied to the Clipboard.
