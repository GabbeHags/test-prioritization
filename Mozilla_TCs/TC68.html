Test Case Description for dragging images across browser windows

Purpose

To verify that dragging an image across browser windows opens the image in the target window.

Steps/Description

  1. Open the testfile.html test file
  2. Open a new browser window
  3. Drag the Space Shuttle image from the first browser window to the target window.

Expected Results

3. The target browser window should open the the space shuttle image
