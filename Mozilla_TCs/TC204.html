Test Case Description for bookmarks links

Purpose

Determine that links in Bookmarks panel work

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. Visit any url, say, www.yahoo.com
  3. Bookmark that link using "Bookmarks | Add Current Page"
  4. Add theBookmarks panel to the Sidebar using the "Customize Sidebar" option
  5. Open the Bookmarks panel in the Sidebar
  6. Double click with the mouse on the bookmark that you just added
  7. The selected link should open in the browser window. Try a few more bookmarks from the panel

Expected Results

Appropriate bookmarks should open in the browser window when clicked from the Bookmarks panel in the Sidebar
