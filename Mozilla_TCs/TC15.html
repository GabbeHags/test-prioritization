Test Case Description for Shortcuts

Purpose

Determine that you can open all the selections in the Help drop down menu using short cut buttons.

Initial Conditions

Netscape should be started up

Steps/Description

Click Alt+H prior to starting each of these steps

  1. Click H
  2. Click W
  3. Click F
  4. Click S
  5. Click P
  6. Click A

Expected Results

1. Help and Support Center page should open (close the help dialog)

2. You should be taken to the Welcome page that describes the new features in Netscape

3. You should be taken to the Feedback page

4. You should be taken to the Security News page

5. You should be taken to the Installed plugins page

6. You should be taken to the About Netscape page
