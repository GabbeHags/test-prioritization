Test Case Description for dragging URL to an AIM window

Purpose

Verify that you can drag a URL or page content to an AIM window

Steps/Description

  1. Minimize a Navigator window and an AIM window so that you can see both
  2. Start an AIM message to an AIM account you control
  3. Go to any webpage that contains a URL in the content
  4. Select the URL in the content and drag to AIM window
  5. Send the AIM message
  6. When the AIM message comes across, verify that the link appears in the message and when clicked, it takes you to the appropriate webpage
