Test Case Description for printing frames

Purpose

Determine that Seamonkey prints pages with frames properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to view a page with horizontal frames
  3. Click here to view a page with vertical frames.
  4. Once each page loads, select menu option "FILE| PRINT"
  5. Press "OK" on Print Dialog Box to print the page.

Expected Results

After Step 5, the page should be printed as seen on screen
