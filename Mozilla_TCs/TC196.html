Test Case Description for default width

Purpose

Determine that default width of Sidebar is proper

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. In the Sidebar, verify that the default width appears properly and the sidebar does not occupy more that 2 inches(approximately) of the width
  3. Quit the browser using File->Quit
  4. Restart the browser

Expected Results

After step 3, the Sidebar default width should remain the same
