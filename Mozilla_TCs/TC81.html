Test Case Description for dragging shortcuts from the desktop

Purpose

Confirm that dragging all types of Internet shortcut to the browser window opens the URL

Steps/Description

Windows

  1. Download and decompress the bookmarks.zip archive file
  2. Drag each shortcut file (from the archives) to the browser window

Mac OS

  1. Download and decompress the bookmarks.sit archive file
  2. Drag each shortcut file (from the archives) to the browser window

Linux

  1. Create a Web shortcut on your desktop environment
  2. Drag the shortcut to the browser window

Expected Results

Windows

Dragging Netcenter (Internet Explorer-originated shortcut, with off-line availability) should open www.netscape.com

Dragging Shortcut to mozilla.org (Communicator 4.7-originated shortcut) should open www.mozilla.org

Mac OS

Dragging www.netscape.com/ (Mac OS Internet Shortcut) should open www.netscape.com

Dragging mozilla.org (Navigator bookmark) should open www.mozilla.org

Linux

The browser should load the shortcut URL
