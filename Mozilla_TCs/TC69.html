Test Case Description for dragging an image onto the desktop

Purpose

To verify that dragging an image onto the desktop saves the image.

Steps/Description

  1. Open the testfile.html test file
  2. Drag the Space Shuttle image onto the desktop

Expected Results

2. The image should be saved onto the desktop.

On Mac OS, the image should also have a file type of JPEG; check using a tool like Resorcerer or ResEdit.)
