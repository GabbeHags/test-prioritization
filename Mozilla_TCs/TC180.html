Test Case Description for printing web pages

Purpose

Determine that Seamonkey prints Top web pages properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to open a listing of top web sites..
  3. Click on the first web page link.
  4. Select the menu option "FILE | PRINT" to print the web page
  5. Press "OK" to print the page
  6. Similarly, try printing all the pages one by one.

Expected Results

All web pages should print properly.
