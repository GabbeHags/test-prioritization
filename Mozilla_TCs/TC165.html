Test Case Description for HTML 4.0 page printing

Purpose

Determine that Seamonkey prints HTML 4.0 pages properly

Initial Conditions

None

Steps/Description

  1. Launch Apprunner
  2. Click here to open a HTML 4.0 page.
  3. Select the menu option "FILE | PRINT" to print the web page
  4. Press "OK" to print the page
  5. Similarly, try printing this page also.

Expected Results

All pages should print properly.
