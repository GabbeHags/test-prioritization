Test Case Description for reordering panels

Purpose

Determine that panels can be reordered

Initial Conditions

None.

Steps/Description

  1. Launch the browser
  2. From the Sidebar , select "Tabs | Customize Sidebar"
  3. From the window that opens up, add a few panels from the left pane
  4. Highlight on any of the panels in the right pane and click on the reorder arrows on the right side
  5. Clicking the reorder arrows should move the panel up/down and you should be able to place them wherever you want (top to bottom)
  6. Clicking "OK" should also reflect the reordering change in the Sidebar in the browser window

Expected Results

You should be able to reorder panels as you want
