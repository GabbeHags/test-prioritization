Test Case Description for printing range

Purpose

Determine that printing range works properly

Initial Conditions

None

Steps/Description

  1. Launch the browser
  2. Go to Windows Build Prerequisites
  3. Do "File | Print"
  4. Once the "Print" dialog comes up, click on the radio button titled "Pages"
  5. Enter a value for the number of pages to print (say 1 to 3)

Expected Results:

Only pages 1 to 3 should get printed
